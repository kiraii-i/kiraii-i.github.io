Guitar Jack (tip)→0.1µF coupling
0.1µF output→Fuzz Q1 Base
Fuzz Q2 output→JRC4558D In+ (OD)
OD Output→LM308 In+ (Dist)
Dist Output→J201 Gate (Boost)
J201 Drain→TL072 In+
TL072 Out→ESP32 GPIO34
📡 TL072 Preamp Bias
5V→R 4.7kΩ #1
R 4.7kΩ #1→TL072 In+ (Bias Point)
R 4.7kΩ #2→GND (Bias divider)
TL072 In−→10kΩ → GND
TL072 VCC→5V
TL072 VEE→GND
الـBias 1.65V يرفع الإشارة من ±0.5V إلى 0–3.3V (مناسب للـADC)
🎵 PCM5102A I2S DAC
ESP32 GPIO22→PCM5102 DIN
ESP32 GPIO25→PCM5102 LCK
ESP32 GPIO26→PCM5102 BCK
3.3V→PCM5102 VCC + AGND tie
GND→PCM5102 GND + FLT(GND)

PCM5102 SCK→GND (master clock off)

خطوات البناء

ابنِ الجهاز خطوة بخطوة

1

اشتري القطع وجهّز مكان العمل

اطبع قائمة القطع وروح المحل. جهّز مكان نظيف، مولتيميتر، ومصدر 9V مؤقت (بطارية أو adapter). ابدأ بفحص كل القطع قبل التوصيل.

2

ابنِ دائرة Power Supply أولاً

وصّل 9V → 7805 → 5V → 78L033 → 3.3V. اختبر بالمولتيميتر قبل توصيل أي IC. أضف مكثفات 470µF و100nF عند كل خرج.

3

وصّل ESP32 + PCM5102 فقط (Bypass Test)

وصّل GPIO22/25/26 للـPCM5102. ارفع كود Bypass بسيط (ADC قرأ → I2S كتب). اختبر: وصّل جيتار في GPIO34 عبر مقسّم الجهد، المخرج في output jack — لازم تسمع الصوت.

4

ارفع الكود الكامل واختبر WiFi

ارفع esp32_pedal_firmware.ino. افتح Serial Monitor — لازم تشوف: WiFi AP: GuitarPedal وWeb server started. وصّل الفون بالـWiFi وافتح http://192.168.4.1.

5

اختبر كل Effect رقمي

من واجهة الفون، جرّب Tremolo أول — أسهل effect تسمعه. بعدين Delay، Chorus، Reverb. لو سمعت التأثير صح، كل حاجة شغالة. اضبط الـParameters من المتصفح.

6

أضف الجزء الأنالوج (Fuzz/OD/Dist)

ابنِ كل circuit أنالوج منفرد وجرّبه. بعدين وصّلهم قبل الـTL072 PreAmp. الـ switches بالـ Bypass تخليك تختار إيهم شغّال. لو في noise، أضف مكثف 100nF على كل VCC.

7

أضف الأزرار الفيزيائية + LEDs

وصّل أزرار NEXT/PREV/BYPASS على GPIO18/19/5. كل button بـPullup داخلي، Pin الثاني على GND. جرّب التنقل بين الـEffects. اضبط WiFi SSID/Password في الكود لو حبيت تغيرهم.
